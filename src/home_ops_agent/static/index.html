Agents

Each agent handles a specific task. Assign cheaper models to simple tasks and more capable models to complex ones.

PR Review

Reviews open Renovate PRs. Reads diffs, checks CI status and labels, posts a review comment with risk assessment. Can push fix commits when auto-merge is enabled.

Alert Triage

First responder for alerts from Alertmanager and Gatus. Checks pod status, reads logs, queries Prometheus metrics, and identifies the root cause.

Alert Fix

Takes corrective action when an issue is found. Can restart pods, reconcile Flux resources, and send enriched diagnostics via ntfy. Every action is logged and reported.

Code Fix

Writes code fixes for failing PRs. Understands Kubernetes manifests, HelmRelease schemas, and can push fix commits to PR branches. Only modifies files under kubernetes/apps/.

Chat

Powers the interactive chat. Answers questions about cluster state, runs diagnostics on demand, and executes commands you ask for. Has access to all tools.
